## 1. VSCode 설치하기

**VSCode가 뭐예요?**
> 코드를 작성하는 메모장 같은 프로그램이에요. 하지만 일반 메모장보다 훨씬 똑똒해요!

### 설치 방법

1. 인터넷 브라우저를 열어요
2. 주소창에 `https://code.visualstudio.com` 입력
3. 파란색 **Download for Windows** 버튼 클릭
4. 다운로드된 파일 실행
5. **다음 > 다음 > 설치** 계속 클릭
6. 설치 완료!

### 확인하기
- 바탕화면이나 시작메뉴에서 **Visual Studio Code** 찾아서 실행
- 파란색 창이 뜨면 성공!

## 2. Node.js 설치하기

**Node.js가 뭐예요?**
> 웹사이트를 만들 때 필요한 도구들을 설치하고 실행하는 프로그램이에요.
> 요리사에게 주방이 필요하듯, 개발자에게 Node.js가 필요해요!

### 설치 방법

1. `https://nodejs.org` 접속
2. **LTS** 버전 다운로드 (왼쪽 초록색 버튼)
   - LTS = 안정적인 버전이라는 뜻
3. 다운로드된 파일 실행
4. **Next > Next > Install** 계속 클릭
5. 설치 완료!

### 확인하기
1. 키보드에서 `Windows 키 + R` 누르기
2. `cmd` 입력하고 엔터
3. 검은 창이 뜨면 `node --version` 입력하고 엔터
4. `v20.x.x` 같은 숫자가 나오면 성공!

## 3. Git 설치하기

**Git이 뭐예요?**
> 코드의 변경 기록을 저장해주는 타임머신이에요.
> 실수해도 과거로 돌아갈 수 있어요!

### 설치 방법

1. `https://git-scm.com` 접속
2. **Download for Windows** 클릭
3. 다운로드된 파일 실행
4. 설치 옵션은 **전부 기본값**으로 Next 클릭
5. 설치 완료!

### 확인하기
1. `Windows 키 + R` → `cmd` → 엔터
2. `git --version` 입력
3. `git version 2.x.x` 나오면 성공!

### Git 초기 설정 (한 번만 하면 됨)
```bash
git config --global user.name "내이름"
git config --global user.email "내이메일@gmail.com"
```

## 4. GitHub 가입하기

**GitHub가 뭐예요?**
> 내 코드를 인터넷에 저장하는 클라우드 창고예요.
> 구글 드라이브가 파일을 저장하듯, GitHub는 코드를 저장해요!

### 가입 방법

1. `https://github.com` 접속
2. **Sign up** 클릭
3. 이메일 입력
4. 비밀번호 만들기
5. 사용자 이름 만들기 (영어로!)
6. 이메일 인증하기
7. 가입 완료!

### GitHub와 내 컴퓨터 연결하기

1. VSCode 열기
2. `Ctrl + Shift + P` 누르기
3. `Git: Clone` 검색해서 선택
4. **Clone from GitHub** 선택
5. GitHub 로그인 허용
6. 연결 완료!

## 5. Vercel 가입하기

**Vercel이 뭐예요?**
> 만든 웹사이트를 인터넷에 올려서 누구나 볼 수 있게 해주는 서비스예요.
> 무료로 웹사이트를 배포(공개)할 수 있어요!

### 가입 방법

1. `https://vercel.com` 접속
2. **Sign Up** 클릭
3. **Continue with GitHub** 선택 (제일 쉬움!)
4. GitHub 연동 허용
5. 가입 완료!

### GitHub 저장소와 연결하기

1. Vercel 대시보드에서 **Add New...** > **Project** 클릭
2. GitHub 저장소 목록에서 내 프로젝트 선택
3. **Deploy** 클릭
4. 잠시 기다리면 웹사이트 주소가 생겨요!

## 6. Claude Code 설치하기

**Claude Code가 뭐예요?**
> AI 비서가 내 컴퓨터에서 직접 코드를 작성해주는 도구예요.
> 말로 설명하면 AI가 코드를 짜줘요!

### 설치 방법

1. `cmd` 창 열기 (`Windows + R` → `cmd`)
2. 아래 명령어 입력:
```bash
npm install -g @anthropic-ai/claude-code
```
3. 설치될 때까지 기다리기 (2-3분)

### Claude 계정 필요해요!
1. `https://claude.ai` 접속
2. 회원가입 또는 로그인
3. Claude Pro 구독 필요 (유료)
   - 또는 API 키 사용 가능

### API 키 받는 법 (개발자용)
1. `https://console.anthropic.com` 접속
2. 로그인
3. **API Keys** 메뉴
4. **Create Key** 클릭
5. 키 복사해서 안전한 곳에 저장

## 7. VSCode에서 Claude Code 연결하기

### 방법 1: 터미널에서 실행

1. VSCode 열기
2. `Ctrl + `` (백틱) 눌러서 터미널 열기
3. 프로젝트 폴더로 이동:
```bash
cd 내프로젝트폴더
```
4. Claude Code 실행:
```bash
claude
```
5. 처음 실행하면 로그인 창이 뜸
6. Claude 계정으로 로그인
7. 연결 완료! 이제 대화할 수 있어요

### 방법 2: VSCode 확장 프로그램 (더 쉬움!)

1. VSCode 왼쪽에서 **확장** 아이콘 클릭 (네모 4개 모양)
2. 검색창에 `Claude` 입력
3. **Claude** 확장 프로그램 설치
4. 왼쪽에 Claude 아이콘 생김
5. 클릭해서 로그인
6. 바로 대화 시작!

## 자주 묻는 질문 (FAQ)

### Q: 설치 중에 오류가 나요!
A: 대부분 **관리자 권한**으로 실행하면 해결돼요.
   - 설치 파일에서 마우스 오른쪽 클릭
   - **관리자 권한으로 실행** 선택

### Q: cmd 창에서 명령어가 안 먹어요!
A: 설치 후 cmd 창을 **새로 열어보세요**.
   기존 창은 새 프로그램을 인식 못 할 수 있어요.

### Q: GitHub 비밀번호를 자꾸 물어봐요!
A: Git 자격 증명 관리자를 사용하세요:
```bash
git config --global credential.helper manager
```

### Q: Vercel에서 배포가 안 돼요!
A: 에러 로그를 확인해보세요. 대부분 원인이 적혀있어요.
   - 환경변수 설정 누락
   - 빌드 명령어 오류
   - 등등...
