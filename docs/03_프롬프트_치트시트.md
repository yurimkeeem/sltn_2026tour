## 효율 팁

### DO (좋은 예)
- 구체적인 수치 제공
- 파일 경로 명시
- 예시/참고 자료 첨부
- 우선순위 표시

### DON'T (나쁜 예)
- "더 예쁘게" (→ 구체적 색상/크기)
- "안 돼요" (→ 현상 + 에러 메시지)
- "전체 다 바꿔줘" (→ 단계별 요청)
- "알아서 해줘" (→ 선택지 제시 요청)
